# 🚀 How to Start Locally

Follow these steps to run the **Agentic AI Mastery Portal** on your local machine.

### Prerequisites
- **Node.js:** (Version 18 or higher)
- **NPM:** (Comes with Node.js)

---

### 💻 For Windows Users
1.  Open your terminal (CMD or PowerShell) in the project folder.
2.  Run the start script:
    ```bash
    start.bat
    ```
3.  Open your browser to: `http://localhost:5173`

---

### 🐧 For Linux/Mac Users
1.  Open your terminal in the project folder.
2.  Run the start script:
    ```bash
    ./start.sh
    ```
3.  Open your browser to: `http://localhost:5173`

---

### 🛠️ Troubleshooting
- **Port already in use?** The server will automatically try the next available port (e.g., 5174).
- **Styling issues?** Refresh the page or run `npm install` again.
